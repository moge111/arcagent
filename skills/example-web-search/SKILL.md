# Web Search Skill

When the user wants to find information online, use this skill.

## Usage
- Fetch search results using the `web_fetch` tool
- Parse and present the most relevant results
- Include source links in responses

## Guidelines
- Prefer authoritative sources
- Present results in a clear, organized format
- Note when information may be outdated
